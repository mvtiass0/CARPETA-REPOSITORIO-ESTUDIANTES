# ARQUITECTURA DEL PENSAMIENTO

## BABBAGE

### Máquina Diferencial
- “La primera computadora del mundo”.
- Máquina mecánica hecha con engranajes.
- Servía para resolver ecuaciones e imprimir tablas matemáticas.
- Automatizaba cálculos que antes se hacían manualmente.

### Máquina Analítica
- Primera máquina de cálculo completamente automática.
- Podía realizar distintos cálculos.

#### Partes
- **The Mill** → CPU / procesador.
- **The Store** → memoria.
- **Punch cards** → software e instrucciones.

---

## JACQUARD

### Telar de Jacquard
- Primer sistema de almacenamiento binario.
- Usaba tarjetas perforadas para automatizar diseños.

#### Sistema binario
- Agujero → **1 / “Encendido”**
- Sin agujero → **0 / “Apagado”**

#### Ideas importantes
- Separación entre **hardware y software**.
- Sistema binario.
- Píxel e imagen rasterizada.

---

## ADA LOVELACE

- **Primera programadora**.
- Trabajó con Charles Babbage y la Máquina Analítica.
- Entendió que la máquina podía trabajar con símbolos, música y arte.

> “La Máquina Analítica teje patrones algebraicos igual que el telar de Jacquard teje flores y hojas”.

### Primer algoritmo
- Entre 1842 y 1843 escribió algoritmos para calcular números de Bernoulli.
- Considerado el primer programa computacional.

---

## TURING

### Máquina de Turing
- Máquina teórica / experimento mental.
- Introduce la idea de computabilidad universal.
- Lo importante pasa a ser el código y las instrucciones.
- Cinta infinita + símbolos + reglas.

## SOFTWARE LIBRE

### Richard Stallman / GNU
- Proyecto GNU → 1983.
- Free Software Foundation → 1985.
- Respuesta a la privatización del software.

### 4 libertades
1. Uso.
2. Estudio.
3. Distribución.
4. Mejora.
